test 1-my_list
=================

for use 'print_sorted' function
---------------------------------

the function have to:

    - print sorted a list


first, import the function:

    >>> MyList = __import__('1-my_list').MyList


examples:


""" whith int """

>>> li = MyList()

>>> li.append(1)
>>> li.append(4)
>>> li.append(2)
>>> li.append(3)
>>> li.append(5)

>>> print(li)
[1, 4, 2, 3, 5]

>>> li.print_sorted()
[1, 2, 3, 4, 5]






""" list whith str and int """

>>> li = MyList()

>>> li.append(1)
>>> li.append(4)
>>> li.append('a')
>>> li.append(3)
>>> li.append(5)

>>> print(li)
[1, 4, 'a', 3, 5]

>>> li.print_sorted()
Traceback (most recent call last):
    ...
TypeError: '<' not supported between instances of 'str' and 'int'







""" all str """

>>> li = MyList()

>>> li.append("b")
>>> li.append("a")
>>> li.append("m")

>>> print(li)
['b', 'a', 'm']

>>> li.print_sorted()
['a', 'b', 'm']




""" empty list """

>>> li = MyList()
>>> print(li)
[]
>>> li.print_sorted()
[]




""" return of list sorted """
>>> li = MyList()
>>> li.append(1)
>>> li.append(3)
>>> li.append(2)
>>>
>>> print(li)
[1, 3, 2]
>>> a = li.print_sorted()
[1, 2, 3]
>>> print(a)
None
